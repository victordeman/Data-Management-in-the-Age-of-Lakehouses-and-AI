Capstone Project

Build Your Own Governed, AI-Ready Open-Source Lakehouse (Weeks 10–15)

Project Overview

100% Linux, zero cloud bills required (runs on laptop or $10-20/month VMs). Required stack: MinIO → Delta/Iceberg → Spark/Polars → Trino → Nessie + OpenMetadata → Airflow + dbt → Feast → Milvus → Ollama RAG demo.

Must demonstrate live:

  • Medallion architecture (bronze/silver/gold)
  • Time travel + branching (Nessie or Delta)
  • Feature serving (online Redis + offline Parquet)
  • Vector search + RAG over your own data
  • Full lineage in OpenMetadata

Step-by-Step Requirements

  1. Set up MinIO for object storage.
  2. Implement Delta/Iceberg tables with Spark/Polars.
  3. Federate queries with Trino.
  4. Add governance via Nessie + OpenMetadata.
  5. Orchestrate with Airflow + dbt.
  6. Serve features with Feast.
  7. Enable vector search with Milvus.
  8. Demo RAG with Ollama.
